## django-auth-app

Sample Django Auth App

* Register new user
* Login user
* Logout user
* Edit profile
* Change password

### Requirements
```
pip
django
virtualenv
```

### Prepare environment
Create virtualenv
```
virtualenv .
```
Activate virtualenv

#### Windows
```
.\Scripts\activate
```

#### Linux
```
./bin/activate
```

### Clone this repository
```
git clone https://github.com/edisona/django-auth-app.git
cd django-auth-app
```

### Run the server
```
python manage.py runserver
```

### Access the app in browser
```
http://127.0.0.1:8000
```
